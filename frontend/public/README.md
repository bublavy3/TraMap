# What is in this directory?

Content that gets copied directly to the root of the build folder.
Therefore, it is accessed like `/tramap-logo.png`, without specifying the public directory.

- `maps` directory contains real maps of public transport network that serve as demo (mocks) for now
- `other` directory contains other real maps that can be used to test creation of maps on image underlay

None of these will be in the final project.
They serve only testing purposes while the project is being developed as part of a diploma thesis.
